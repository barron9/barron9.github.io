Actor-Critic: One-Step Update for Actor and Critic Networks

In Actor-Critic reinforcement learning methods, there are two main components: the Actor (policy network) and the Critic (value network). These two networks work together to improve the agent’s performance. Let’s break down the one-step update for both networks.

1. Actor Network - One-Step Update

The Actor network is responsible for determining which action to take based on the current state. It generates a policy which is typically represented as a probability distribution over actions in a given state.

One-Step Update for Actor:

The actor adjusts its policy based on feedback from the Critic. This feedback is provided by the Temporal Difference (TD) error, which is calculated by the critic. The TD error reflects how well the actor performed in a given state-action pair.

TD Error Calculation:

The TD errorδtis calculated as follows:

δt=rt+1+γV(st+1)−V(st)

Where: -rt+1 is the reward received after taking actionat, -V(st) is the value estimate of the current statest, -γ is the discount factor, -V(st+1) is the value estimate of the next statest+1.

Policy Gradient Update:

The actor updates its policy by adjusting the policy parameters θactor. This update is based on the TD error:

θactor←θactor+αδt∇θlog⁡πθ(at|st)

Where: -αis the learning rate, -δtis the TD error (from the critic), -∇θlog⁡πθ(at|st)is the gradient of the log-probability of the action taken under the policy.

This update aims to:

  • Increase the probability of actions that lead to higher-than-expected rewards (positiveδt),
  • Decrease the probability of actions that lead to lower-than-expected rewards (negativeδt).

2. Critic Network - One-Step Update

The Critic network estimates the value function (either state-valueV(st)or action-valueQ(st,at)) to evaluate the action taken by the actor. The critic’s job is to provide feedback on how good the action taken by the actor was.

One-Step Update for Critic:

The critic evaluates the accuracy of the actor’s decision by calculating the TD error. This error reflects how much better or worse the action was, based on the observed reward and the value of the next state.

For state-value functionV(s), the TD error is:

δt=rt+1+γV(st+1)−V(st)

For action-value functionQ(s,a), the TD error would be:

δt=rt+1+γQ(st+1,at+1)−Q(st,at)

Critic Update Rule:

  • For state-value functionV(s), the update rule for the critic is:

V(st)←V(st)+βδt∇θV(st)

  • For action-value functionQ(s,a), the update rule is:

Q(st,at)←Q(st,at)+βδt∇θQ(st,at)

Where: -β is the learning rate for the critic, -δt is the TD error (from the actor’s feedback), -∇θV(st) or ∇θQ(st,at) is the gradient of the value function with respect to the parameters.

Summary of One-Step Updates:

Component One-Step Update
Actor The actor adjusts its policy parametersθactorusing policy gradients based on the TD error δt provided by the critic. The update aims to increase the probability of good actions and decrease the probability of bad actions.
Critic The critic updates its value function (eitherV(s) or Q(s,a)) using the TD errorδt. This feedback helps the critic improve its value estimates over time.

Example of One-Step Update:

Consider an agent navigating an environment:

  1. Actor selects action at in state st.
  2. The environment provides a reward rt+1 and the next state st+1.
  3. The Critic calculates the TD error:

δt=rt+1+γV(st+1)−V(st)

  1. The Actor updates its policy using the TD error. Ifδt>0, the policy is adjusted to favor the action taken. Ifδt<0, the policy is adjusted to avoid that action.
  2. The Critic updates its value estimate of the statest(orst,at) to minimize the TD error.

Key Points:

  • Actor Network:

    • Uses the TD error to update the policy.
    • Adjusts the action probabilities to improve reward-maximizing behavior.

  • Critic Network:

    • Uses the TD error to update the value function.
    • Aims to improve the accuracy of value predictions, helping the actor learn from better feedback.

The Actor-Critic method involves simultaneous updates to both networks: the actor learns to take better actions based on the critic’s value function, and the critic improves its value estimates based on the actions taken by the actor.
